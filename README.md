# 沖縄県議会 議員・会派名 変換辞書データ

## 概要

このリポジトリは、**沖縄県議会議員の氏名**および**会派名**を、より正確かつスムーズにかな漢字変換するための辞書データです。
主に **Google 日本語入力（または対応するIME）** での利用を想定して作成されています。

## 使い方（Google 日本語入力へのインポート手順）

この辞書データを導入することで、議員名を簡単に入力できるようになります。

### 1. データのダウンロード

以下のリンクから辞書データファイル（TSV形式）をダウンロードしてください。

* [**`okinawa_gikai_names.tsv` をダウンロード**](https://github.com/koj911/okinawaken-gikai-dictionary/raw/main/data/okinawa_gikai_dictionary.txt)

### 2. Google 日本語入力でのインポート

お使いの環境に応じて以下の手順でインポートします。

1.  **環境設定を開く:**
    * キーボード右下の入力モードを右クリックし、「**辞書ツール**」を開きます。
2.  **辞書をインポート:**
    * 辞書ツールウィンドウで「**管理**」メニューから「**新規辞書にインポート**」を選択します。
3.  **ファイルを選択:**
    * ダウンロードした `okinawa_gikai_dictionary.txt` ファイルを選択します。
4.  **辞書名を設定:**
    * 新しい辞書の名前（例:「沖縄県議会名簿」など）を設定して完了です。

### 3. 利用開始

インポート後、IMEで議員名や会派名の**読み**を入力するだけで、候補に正確な漢字表記が表示されるようになります。

辞書ツールを開き、メニューバー -> 管理 -> 新規辞書にインポート(あるいは 選択した辞書にインポート)
